# clean-code
클린코드 서적 읽기 스터디입니다.

### 스터디 규칙
- 시간: 매주 목요일 10시~11
- 분량: 매주 두장
- 장소: 디스코드
- 스터디 시작할 때 발표자 랜덤 지정
- 읽으면서 내용에 해당되는 예시 본인 프로젝트에서 찾기 혹은 예시 만들기
- 읽은 내용 정리해서 해당 Repository에 PR
    - 업로드 위치: /clean-code/n장/${my_username}/${my_username}.md
